# Project Slytherin

Project Slytherin is a comprehensive Capture The Flag (CTF) challenge designed to test and enhance cybersecurity skills. This repository contains various assets, tools, and resources for participants to engage in penetration testing, cryptographic challenges, and network analysis.

## Project Structure

- **ctf_schedule.pdf**: The schedule for the CTF competition.
- **pentest_report.pdf**: A sample penetration testing report.
- **rulebook.pdf**: The official rulebook for the competition.
- **clues/**: Contains various files and scripts for solving challenges, including encrypted files, logs, and key generators.
- **ctf_compition_assets_and_screenshots/**: Includes screenshots, intercepted data, and other assets related to the competition.
- **deployed_website_for_pentest/**: Contains the setup for a deployed website used for penetration testing, including Docker configurations and application files.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Manjushwarofficial/Project-Slytherin.git
   cd Project-Slytherin
   ```

2. Review the rulebook (`rulebook.pdf`) to understand the competition guidelines.

3. Explore the `clues/` directory for challenge files and scripts.

4. Set up the penetration testing environment using the `deployed_website_for_pentest/` directory:
   ```bash
   cd deployed_website_for_pentest
   docker-compose up
   ```

## Prerequisites

- Docker and Docker Compose
- Python 3.x
- Tools for analyzing PCAP files, encrypted data, and logs

## How to Use

1. Navigate through the directories to find challenge-specific files.
2. Use the provided scripts and tools to solve challenges.
3. Submit your solutions via the competition website.
